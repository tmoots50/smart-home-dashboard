# pi-wall-tablet

## Status
Active — started 2026-04-20. Target completion during paternity leave (~2026-06-07).

## Mission
Build an elegant, genuinely useful wall-mounted tablet with custom Raspberry Pi hardware, and write the apps that run on it during paternity leave.

## Scope
- In scope: hardware sourcing + assembly (Pi, display, mount, power), custom UI apps (home dashboard, family photos, calendar, baby-data, weather), OS + software stack decisions, app code, installation.
- Out of scope: generic Pi tutorials not specific to the wall-mount use case, off-the-shelf smart-display alternatives (this is explicitly a build project).

## Key files
- `_context/` — project-specific context (hardware BOM, design notes, decisions)

## Success criteria
- Tablet mounted on wall and in daily use
- At least two custom apps written by Tim (vs. off-the-shelf) running on it
- Documented enough that a second one could be built from this repo
